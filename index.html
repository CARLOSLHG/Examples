Dibujando en Carboncillo

Prueba nuestras técnicas de diseño en carboncillo

1. Difuminado:

Usado para suavizar sombras y transiciones. Se utiliza el dedo, un paño o un difumino para mezclar el carboncillo y crear degradados suaves.

2. Sombreado:

Aplicar capas de carboncillo en diferentes intensidades para crear volumen y profundidad. Se puede hacer con trazos suaves o líneas más intensas.

3. Borrado selectivo:

Se utiliza una goma de borrar para crear luces y resaltar áreas específicas, como reflejos o zonas iluminadas.

4. Técnica de línea:

Usar trazos de líneas definidos para crear texturas y detalles finos, como en los cabellos o en los contornos de un retrato.

5. Aplicación de contraste:

Crear dramatismo jugando con los extremos del blanco y negro, destacando zonas muy oscuras y claras para acentuar los detalles.
